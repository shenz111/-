# course
本文基于用户的使用场景，设计和实现了基于三端系统的学生智能选课管理系统，本系统基于 Spring Boot 框架构建后端服务和基于 Vue 框架构建前端接口,整体采用 SpringMVC 实现分层架构设计，用 MySQL 关系型数据库进行结构化数据存储，同时通过 Maven 进行程序构建和管理。为了更加高效地处理用户头像文件，通过时间方法生成唯一标识，结合文件流读写与自定义钩子函数，实现图片上传、存储及动态渲染的一体化处理。在角色权限的控制层面，基于前端条件渲染技术实现角色权限的动态分配。选课功能集成多层判断机制。通过数据库中关联查询实时更新课程人数，前端通过组件式开发动态控制按钮状态，并且集成基于推荐算法的课程推荐模块通过分析课程热度生成个性化推荐列表。
